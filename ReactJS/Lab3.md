## Create new project "lab3"
```
$ npx create-react-app lab3
```

## Install [React Router DOM](https://www.npmjs.com/package/react-router-dom)
```
$ npm i react-router-dom
```

## Install Bootstrap
```
$ npm i bootstrap
```

## Install jQuery
```
$ npm i jquery
```

## Install json-server
```
$ npm i json-server -g
```

## Start 
```
$ npm start
```
```
$ json-server --watch --port 3003
```

## Commands
- Install VueJS
```
npm i -g @vue/cli
```

- Create new project "lab1"
```
vue create lab1
```

- Run 
```
cd lab1
```
```
npm run serve
``` 

- Install Packages 
```
npm i
```
```
npm i bootstrap jquery popper
```
```
npm i axios
```
```
npm i vue-router@4
```

## Commands
- Install VueJS
```
$ npm i -g @vue/cli
```

- Create new project "lab3"
```
$ vue create lab3
```

- Install Packages 
```
$ npm i 
```

- Run 
```
$ cd lab3
```
```
$ npm run serve
```
